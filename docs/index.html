Локальне шифрування

Local Encryption

Жоден байт тексту не зберігається у відкритому вигляді. Усе шифрується прямо у вашій оперативній пам'яті.

Not a single byte of text is stored in plain text. Everything is encrypted directly in your RAM.

Zero-Knowledge

Zero-Knowledge

Наш сервер — сліпий. Ми бачимо лише набір випадкових символів і не маємо доступу до ваших паролів.

Our server is blind. We only see random characters and have no access to your passwords.

Відкритий код. Жодних секретів від вас.

Open Source. No secrets from you.

Ми віримо, що справжня безпека не може бути закритою. Весь вихідний код клієнта та сервера доступний для аудиту кожному.

We believe true security cannot be closed-source. All client and server source code is available for anyone to audit.
